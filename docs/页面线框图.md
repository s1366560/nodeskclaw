# 页面线框图

## Portal 实例访问令牌改造

本次只调整 Portal（用户门户）中的两个页面：

- `nodeskclaw-portal/src/components/workspace/AgentDetailDialog.vue`
- `nodeskclaw-portal/src/views/InstanceDetail.vue`

### 改造目标

- 删除原有 `DeskClaw 访问地址` 展示区
- 删除原有“打开实例”按钮
- 改为展示 `访问令牌`（访问 Token）
- 令牌默认脱敏显示，中间以固定掩码替换
- 保留“复制令牌”能力
- 新增“重设令牌”按钮
- 点击“重设令牌”前必须弹出 `Dialog`（确认弹窗），明确说明会自动重启实例

### 工作区弹窗

位置：`AgentDetailDialog.vue`

#### 弹窗当前

- 卡片标题：`DeskClaw 访问地址`
- 副说明：点击即可打开 AI 员工
- 操作：打开 / 复制完整 URL

#### 弹窗改造后

- 卡片标题：`访问令牌`
- 副说明：
  - 正常态：用于连接当前 AI 员工，复制后可手动粘贴使用
  - 重启态：实例正在重启，请稍候再使用新令牌
- 内容区：
  - 单行等宽字体展示脱敏令牌，例如 `ab12cd••••••••••••ef34`
  - 右侧放“复制”按钮
- 操作区：
  - `重设令牌` 按钮
  - 重启中时按钮禁用并显示 loading（加载中）态

#### 重设确认弹窗

- 标题：`重设访问令牌`
- 说明文案：
  - 将生成新的访问令牌
  - 旧令牌会立即失效
  - 实例会自动重启
  - 重启期间服务将短暂不可用
- 按钮：
  - 取消
  - 确认重设

### 实例详情页

位置：`InstanceDetail.vue`

#### 详情页当前

- 卡片标题：`DeskClaw 访问地址`
- 操作：打开 / 复制完整 URL

#### 详情页改造后

- 与工作区弹窗保持一致：
  - 展示脱敏后的访问令牌
  - 提供复制令牌按钮
  - 提供“重设令牌”按钮
  - 使用确认弹窗说明“会生成新令牌并自动重启实例”

### 交互约束

- 前端不再展示完整 URL
- 前端不再提供“打开实例”入口
- 前端复制的是原始令牌，不是脱敏后的字符串
- 重设成功后立即进入 `restarting`（重启中）状态提示，并沿用现有轮询逻辑等待实例恢复
- 两个页面的文案必须接入 i18n（国际化）
